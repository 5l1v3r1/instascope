# Install
-   download [python 3.7 or high](https://python.org/download)
-   install dependencies:
    -   `pip install saya`
    -   `pip install pillow`
    -   `pip install beautifulsoup4`
-   clone this repo: `git clone https://github.com/Ythosa/instascope`
-   run: `python -m main.py`
